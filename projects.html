Programming Projects

Traveling Salesman Problem

Implemented the arbitrary insertion heuristic to quickly find a path. Using this path and cost, I created another function that generates permutations on this path. With the branch and bound algorithm and Prim's algorithm for creating a minimum spanning tree, I found the most optimal path that minimizes path cost.

Relational Database Emulator

Implemented a simplified version of SQL to provide an interface for efficient data retrieval. Also efficiently optimized runtime and memory tradeoffs for various database functions such as retrieval, table joins, and data insertion using various data structures such as hash tables.

Piazza Post Classifier

Implemented a program that analyzes the content of student Piazza posts to classify them into categories by calculating the logarithmic probability of posts to belong to certain categories to automatically classify. Also could also predict never-seen before posts through machine learning and training the algorithm with datasets.

Image Resizing Via Seam Carving

Implemented a program that can resize input image files through seam carving to output edited images. Mathematically calculated the most optimal and cheapest seam to remove by comparing the costs to remove the lowest total energy pixels.

Statistics Calculator

Implemented a calculator that reads in files to rapidly output various statistics regarding the data read. Capable of reading in any data spreadsheet converted to .tsv files to efficiently read in large columns of data to easily keep track of statistical computations.

Battleship Simulator

Implemented a program that simulates the classic boardgame Battleship.

Calorie, Macro, and Nutrient Tracker

Implemented a program that allows the user to log and track their caloric intake, calories burned, and macros consumed. Can also log prior food choices and recommend suggestions based on nutritional value.

Stock Market Simulator

Implemented a program that uses priority queues to determine the best time to buy and sell stocks. Analyzes and pinpoints stocks at their lowest price to buy, as well as their highest price to sell using both a stack and queue.
